Detailed Description

This class is responsible for the AI of the game connect Four its responsibility is to find the next move on the board. it does it by applying the MCTS-algorithm.

Note
there are 6 main parameters which can be adjusted: 
      MAX_NUM_OF_ITERATIONS
      VISITS_TO_START_EXPAND_ALL
      Value::WIN, Value::DRAW, Value::LOOSE
      CURIOUSITY_FACTOR C
      * small C = > game tree gets deeper expanded
        (only the best variation gets explored)
      * big C = > game tree gets broader expanded
        (nodes with lesser visits are prefered)

for more information on the mcts, see: http://de.slideshare.net/ftgaic/mcts-ai https://en.wikipedia.org/wiki/Monte_Carlo_tree_search rewards / ratings taken from http://www.tantrix.com/Tantrix/TRobot/MCTS%20Final%20Report.pdf
